The client-side.js file initializes the variable buttonClicks when the page loads. Each time the button is clicked, buttonClicks is incremented. Next, the client sends a message to the server to inform it that the button was clicked. The server also has a variable for tracking the number of times the button was clicked. The server sends it's button click count in the response.

  1. Open the browser console to see the logs.
  2. Click the button multiple times.
  3. Do the server and client counts match?
  4. Reload the page in the browser.
  5. Click the button multiple times.
  6. Do the server and client counts match?
